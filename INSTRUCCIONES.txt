========================================
  CONTENTFORGE - PLAN DEL PROYECTO
========================================

OBJETIVO
--------
App web local (localhost) que genera imágenes estáticas
para TikTok (carruseles, frases, anuncios) usando
Python + Flask + Pillow.

PRINCIPIO MODULAR
-----------------
- Cada archivo tiene UNA sola responsabilidad.
- PlantillaBase = motor de dibujo puro. Crea imagen + dibuja texto centrado.
  No sabe de archivos ni carpetas.
  Método público: _crear_imagen(texto) → retorna PIL.Image
- Carrusel = clase principal. Hereda de PlantillaBase.
  __init__ procesa texto crudo → lista de slides.
  generar() crea carpeta, llama _crear_imagen() por cada slide, guarda.
- Quote y Announcement son Carrusel con 1 slide (misma mecánica).
- Output simple: cada generación crea su carpeta con timestamp dentro de output/.

PRUEBAS UNITARIAS
-----------------
- Framework: pytest (pip install pytest)
- Cada módulo tiene su archivo de test en tests/
- Se corren con: pytest tests/ -v
- Tests se escriben DESPUÉS del código de cada módulo
- Ejemplo:
  Día 2: creás colors.py → escribís test_colors.py
  Día 3: creás carousel.py → escribís test_carousel.py
