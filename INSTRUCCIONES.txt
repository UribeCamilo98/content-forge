========================================
  CONTENTFORGE - PLAN DEL PROYECTO
========================================

OBJETIVO
--------
App web local (localhost) que genera imágenes estáticas
para TikTok (carruseles, frases, anuncios) usando
Python + Flask + Pillow.

PRINCIPIO MODULAR
-----------------
- Cada archivo tiene UNA sola responsabilidad.
- PlantillaBase = motor de dibujo puro. Crea imagen + dibuja texto con
  fuente, tamaño y alineación configurables. No sabe de archivos ni carpetas.
  Método público: _crear_imagen(texto, fuente, tamano, alineacion) → retorna PIL.Image
- Carrusel = clase principal. Hereda de PlantillaBase.
  __init__ procesa texto crudo → lista de slides.
  generar() crea carpeta, llama _crear_imagen() por cada slide, guarda.
  Recibe y pasa parámetros de fuente, tamaño y alineación.
- Carrusel maneja N slides (1-20), cubre casos de quote/anuncio.
- FontManager = escanea fonts/ y provee lista de fuentes disponibles. ✅
- ImageManager = administra sets de imágenes en images/.
- LayoutEngine = aplica plantillas predefinidas (texto+imágenes).
  Método público: aplicar(texto, imagenes, template) → dict de posiciones
- VisualEditor (v1.2) = canvas JS en frontend para edición en vivo.
- Output simple: cada generación crea su carpeta con timestamp dentro de output/.
